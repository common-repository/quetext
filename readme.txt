Quetext’s Wordpress Plugin uses DeepSearch™ technology to identify any content throughout your work that might be plagiarized.

== Installation ==

1. Upload plugin file on add plugin section of your Wordpress website
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You need to log into plugin before using it
4. To create an account, please visit https://www.quetext.com/pricing

== Use of 3rd party/main app services ==

We make use of services available at QueText (www.quetext.com). These services include:
1. https://www.quetext.com/api/auth/login for authentication
2. https://www.quetext.com/api/wordpress/download-report to download report
3. https://www.quetext.com/api/wordpress/check-status to check plagiarism status
4. https://www.quetext.com/api/wordpress/report - get plagiarism report
5. JQuery knob 1.2.12 (https://plugins.jquery.com/knob)
6. Fontawesome Kit (https://fontawesome.com/v6/download)
7. Boostrap 5.2 (https://getbootstrap.com/docs/5.2/getting-started/download)
